# Dzawani Tour

> Your Colorful Journey Starts Here

Website resmi **Dzawani Tour** — platform wisata terpercaya yang menawarkan paket tour domestik, internasional, dan umrah terbaik di Indonesia.

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Deployment**: Vercel

## Brand Colors

| Color | Hex | Usage |
|-------|-----|-------|
| Dzawani Magenta | `#e85297` | CTA Buttons, Primary Actions |
| Royal Purple | `#503861` | Headers, Brand Elements |
| Ocean Blue | `#00BBF9` | Brand Identity, Links |

## Fonts

- **Poppins** — Headings & Titles
- **Inter** — Body Text
- **Montserrat** — Special Elements

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

## Project Structure

```
src/
├── app/                    # Next.js App Router pages
│   ├── page.tsx           # Homepage
│   ├── paket/             # Tour packages
│   ├── destinasi/         # Destinations
│   ├── blog/              # Blog
│   ├── tentang/           # About us
│   └── kontak/            # Contact
├── components/
│   ├── layout/            # Navbar, Footer
│   ├── sections/          # Page sections
│   └── ui/                # Reusable UI components
├── lib/                   # Data & utilities
└── types/                 # TypeScript types
```

## Deployment

Deployed on [Vercel](https://vercel.com) with automatic deployments from the `main` branch.
